
### 目录结构

```
.
├── README.md
└── src
    ├── css
    ├── index.html
    ├── js
    │   └── hello.js
    ├── lib
    │   └── require.js
    ├── main.js
    └── util
```

- `css`：样式文件存放目录
- `js`：JavaScript文件存放目录
- `lib`：类库、JavaScript框架、css框架存放目录
- `util`：JavaScript工具类存放目录
- `index.html`：项目首页
- `main.js`：require.js全局配置文件
